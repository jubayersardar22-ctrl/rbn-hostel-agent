# 🏠 নিবেদিকা ভিআইপি হোস্টেল - WhatsApp এজেন্ট

## 📖 পরিচিতি

নিবেদিকা ভিআইপি হোস্টেলের জন্য সম্পূর্ণ WhatsApp AI এজেন্ট। এই এজেন্ট:
- ✅ হোস্টেল সম্পর্কিত সকল প্রশ্নের উত্তর দেয় (বাংলায়)
- ✅ রুমের ছবি পাঠায়
- ✅ মেইনটেন্যান্স সমস্যা সংশ্লিষ্ট স্টাফকে ফরওয়ার্ড করে
- ✅ ২৪/৭ চালু থাকে

## 🚀 চালু করার নিয়ম

### ধাপ ১: Node.js ইনস্টল করুন
https://nodejs.org থেকে LTS ভার্শন ডাউনলোড করুন

### ধাপ ২: Dependencies ইনস্টল করুন
```bash
npm install
```

### ধাপ ৩: এজেন্ট চালু করুন
```bash
npm start
```

### ধাপ ৪: QR Code স্ক্যান করুন
- Terminal এ QR Code দেখাবে
- WhatsApp খুলুন
- ⋮ (তিনটি ডট) > Linked Devices > Link a Device
- QR Code স্ক্যান করুন

✅ **ব্যস! এজেন্ট চালু হয়ে গেছে!**

| `৩` বা `শাখা` | শাখার ঠিকানা |

| `৬` বা `ছবি` | রুমের ছবি |

## 🔧 স্টাফ নম্বর পরিবর্তন

`.env` ফাইলে নম্বর পরিবর্তন করুন:

```
ELECTRICIAN_NUMBER=01XXXXXXXXX  # বিদ্যুৎ সমস্যার জন্য
PLUMBER_NUMBER=01XXXXXXXXX      # পানির সমস্যার জন্য
COOK_NUMBER=01XXXXXXXXX         # খাবারের সমস্যার জন্য
```

## 📸 রুমের ছবি যোগ করুন

`images/` ফোল্ডারে ছবি রাখুন (JPG, PNG, WEBP ফরম্যাট):
```
images/
├── room_1.jpg
├── room_2.jpg
├── dining.jpg
└── bathroom.jpg
```

## 🔄 মেইনটেন্যান্স সিস্টেম

বোর্ডার যখন সমস্যা রিপোর্ট করে, এজেন্ট স্বয়ংক্রিয়ভাবে:
1. সমস্যার ধরন শনাক্ত করে (বিদ্যুৎ, পানি, খাবার, ইত্যাদি)
2. সংশ্লিষ্ট স্টাফকে WhatsApp মেসেজ পাঠায়
3. বোর্ডারকে কনফার্মেশন দেয়

## 📂 ফাইল কাঠামো

```
nibedika-whatsapp-agent/
├── index.js              # মূল এজেন্ট ফাইল
├── knowledge_base.js     # সব তথ্য এখানে
├── handlers/
│   ├── messageHandler.js # মেসেজ প্রসেসিং
│   ├── mediaHandler.js   # ছবি পাঠানো
│   └── maintenanceHandler.js # সমস্যা ফরওয়ার্ড
├── images/               # হোস্টেলের ছবি রাখুন এখানে
├── .env                  # স্টাফ নম্বর কনফিগারেশন
└── package.json
```

## ⚠️ সমস্যা হলে

1. `node_modules` মুছুন এবং `npm install` দিন
2. `.wwebjs_auth` ফোল্ডার মুছে আবার QR স্ক্যান করুন
3. Chrome/Chromium ইনস্টল করুন
